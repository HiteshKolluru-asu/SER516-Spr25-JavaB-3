# frontendvue

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

```
npm install -g serve
```
### Serve the dist folder
```
serve -s dist
```

npm run lint
